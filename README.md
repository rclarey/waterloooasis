# Getting set up

- Install node, if necessary:
- Run `npm install`
- Run `sudo npm i -g grunt`
- Run `grunt symlink`
- In two terminal tabs, run:
    - `grunt`, and
    - `JWT_SECRET=<your jwt secret> COOKIE_SECRET=<your cookie secret> grunt web`
